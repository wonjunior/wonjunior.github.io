Publications

Material Transforms from Disentangled NeRF Representations

Ivan Lopes, Jean-François Lalonde, Raoul de Charette
Preprint (arXiv), 2024

A method to transfer material transformations across scenes. Building on disentangled NeRF representations, our approach learns to map Bidirectional Reflectance Distribution Functions (BRDF) from pairs of scenes observed in varying conditions.

arXiv Code

Material Palette: Extraction of Materials from a Single Image

Ivan Lopes , Fabio Pizzati, Raoul de Charette
Conference on Computer Vision and Pattern Recognition (CVPR), 2024

Material Palette extracts a palette of PBR materials (albedo, normals, and roughness) from a single real-world image using diffusion priors.

Project Page PDF arXiv Code

DenseMTL: Cross-task Attention Mechanism for Dense Multi-task Learning

Ivan Lopes, Tuan-Hung Vu, Raoul de Charette
Conference on Applications of Computer Vision (WACV), 2023

This work proposes an architecture to perform multi-task learning by combining information from different tasks using correlation-guided and self-attention mechanisms.
